# вҡҷ Installation

## Clone the Repository

```bash
git clone https://github.com/Greeshmita-23/portfolio-website.git
```

## Navigate to Project

```bash
cd portfolio-website
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```
